## What's New in v0.3.0

### New Components
- **Token Rotator**: Dex password grant вҶ’ rotated spoke kubeconfigs every 5m; metrics on `:8080/metrics`
- **Rotating Trust**: Dex `trust-jwks` ConfigMap + oidc-verifier key merging; credential rotation without controller restarts
- **Multi-Tenancy**: `RegionalWidgetRequest.spec.tenant.id` вҶ’ namespace-scoped Widget creation on spoke; per-tenant RBAC

### New Tests (5 added, 10вҶ’15)
- `11-rotating-trust`: Dex trust key bootstrapping and mounting
- `12-dashboard-metrics`: Prometheus data validation for binding-controller + token-rotator
- `13-tenant-isolation`: Tenant-scoped Widget routing and negative assertion
- `14-token-rotation-proof`: End-to-end token rotation verification with metrics
- `15-dashboard-data`: Grafana panel data rendering via PromQL

### Infrastructure Changes
- Flux CD HelmRelease now enables `ingress.enabled: true` for GitOps-deployed configurations
- `RegionalWidgetRequest` CRD extended with `tenant` object property
- 4 new Grafana dashboards: Chainsaw Results, Cluster Fitness, Controller Deep Dive, Token Rotation
